What is this page?

A list of upcoming features and their status!

Not every feature is ready to use, but since the ticket system is functional, Engineering wanted to roll it out and get started.

If you notice a bug or issue in the website, or if you want a feature added, and it's not already listed here, submit a new ticket! Use the "Ticket System" category and this page will be updated to show it's status.

Upcoming Features

    Currently, tickets are meant to be viewed from the profile page only. The ability to assign users and update the ticket from the profile page is in progress.
    In Progress

    Unscheduled
